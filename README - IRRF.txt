
README - DESCRIÇÃO DA PLANILHA

===============================
Visão Geral
===============================
Esta planilha foi desenvolvida para coletar, organizar e analisar dados financeiros pessoais, incluindo dados do titular, informes bancários, notas bancárias (entradas) e uma lista de bancos utilizados. Ela é estruturada em quatro abas principais:
- TITULAR
- INFORMES
- NOTAS
- TABELAS

===============================
1. Aba TITULAR
===============================
Contém informações pessoais do titular da planilha.

Campos:
- NOME: Nome completo do titular.
- CPF: Cadastro de Pessoa Física.
- NASCIMENTO: Data de nascimento.
- TÍTULO DE ELEITOR: Número do título de eleitor.
- CÔNJUGE: Nome do cônjuge, se aplicável.
- ENDEREÇO: Rua, número e outros detalhes do endereço.

Como funciona:
O usuário preenche os dados diretamente na planilha. Esses dados são usados para identificar o titular e podem ser referenciados em outras abas.

===============================
2. Aba INFORMES
===============================
Coleta dados de rendimentos bancários e lista bancos.

Campos:
- TOTAL: Soma de todos os valores declarados em cada banco.
- BANCO: Código e nome do banco (ex.: "336 - C6 Bank").
- VALOR ATUAL: Valor atualizado do rendimento bancário.

Como funciona:
O usuário insere o valor total e, depois, detalha cada banco e seu valor.
Os dados são digitados manualmente; não há fórmulas automáticas visíveis.
Os códigos de banco são listados para facilitar a padronização e organização.

===============================
3. Aba NOTAS
===============================
Destinada ao controle das entradas mensais, como holerites e extratos.

Campos:
- DATA: Data da entrada bancária.
- CATEGORIA: Tipo de entrada (ex.: "HOLERITE").
- VALOR: Valor financeiro recebido.

Como funciona:
O usuário insere manualmente as datas, categorias e valores.
Esses dados poderiam ser utilizados para análises de fluxo de caixa ou conciliação com bancos.

===============================
4. Aba TABELAS
===============================
Lista auxiliar com nomes e códigos de bancos.

Exemplo de dados:
- 1 - Banco do Brasil
- 102 - XP Investimentos CCTVM S.A.
...

Como funciona:
Tabela de apoio usada para padronizar nomes de bancos e seus códigos.

===============================
Itens Atrelados
===============================
- Documentos fiscais: é recomendado vincular ou anexar PDFs dos holerites ou extratos mensais aos lançamentos na aba NOTAS.
- Conciliação bancária: ao preencher valores na aba INFORMES, seria interessante comparar com as entradas na aba NOTAS.
- Controle anual: Futura criação de uma aba adicional para relatórios anuais consolidados.
